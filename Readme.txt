




Sign in to the website. After logging in, you will be directed to the home page, which contains all the public and private playlists associated with the logged-in user.
In the playlists you can view movies that are added to the list and you can even remove movie from the list through remove option available

By clicking on "Public Playlists" in the navbar, you can see all the public playlists available on the website.As well you can movie all the movies in the playlist but you cannot edit the playlists as they are others public playlists

Using the search option, you can retrieve a movie. You will see a "+" icon next to the movie, where you can add the movie to your playlist or create a new playlist and add the movie to it.

You can also logout at any time.
